
# Android Marketplace Application

![Tech](https://img.shields.io/badge/Stack-Android%20Kotlin%20%7C%20Firebase%20%7C%20RESTAPI-orange)

A modern Android marketplace app with real-time order tracking, vendor catalog, and in-app chat.

## Features
- Real-time chat (WebSockets / Firebase)  
- Product listing and vendor pages  
- Cart + checkout  
- Order tracking (Driver → Vendor → Customer)  
- Secure REST API backend  
- Firebase push notifications  

## Architecture
- **Frontend:** Android (Kotlin)  
- **Backend:** Laravel API  
- **Realtime:** Firebase  
- **Database:** MySQL   

## Installation
Import the project in **Android Studio**, sync Gradle, and run.

## API Configuration
Update API URL in `app/src/main/java/.../Config.kt`.
